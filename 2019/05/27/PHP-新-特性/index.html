php＂新＂特性总结

虽然之前写的项目都上了PHP7.1，但是没有用到PHP的新特性，还是维持PHP5.6的写法．模模糊糊了解过PHP的新特性比如说限制参数类型，限制返回参数限制，??，<=>等．但是刷了EasySwoole的文档后，很多特性看不懂，特地重新归纳了一遍特性．并且以后都用上新特性．

PS：这里的＂新＂特性，不单单指PHP7新特性，还有我新学到的特性用法

  • 7.0

    1. 标量类型声明　添加 int string bool float ．( callback object array 等已在5.6中添加了 )

    2. 返回类型声明　类型如标量声明类似

    3. NULL合并符（??）

    4. use

    5. define可以定义数组 ( 5.6 const可以定义数组)

  • 7.1

    1. 可为空类型

      1. 新返回类型 void

      2. 字符串支持负偏移量 (我一直以为早就有了没想到7.1才有,一直在用)

      3. 常量可见性

      4. callables可以转成闭包(Closure::fromCallable())　

        没有短闭包之前,感觉都不太实用．虽然从7.0就不断扩充Closure的方法

      5. list 支持键名并且可以用 [] 缩写

      6. 多异常捕获

  • 7.2

    1. 新的对象类型　(意思不大 感觉没啥用)
    2. 允许抽象类继承抽象类的时候，重写抽象方法
    3. 允许实现接口时，省略参数类型
    4. 允许命名空间分组，最后使用逗号

  • 7.3

    1. 允许引用list

    2. 函数可以尾随逗号
